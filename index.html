Roguelike Dungeon Generator

Dungeon Generator was written solely by Trevor Evans, meant to be a design test for a later rewrite in C++ for use in the Unreal 4 game engine. It implements the design scheme laid out by Bob Nystrom, found here, with some additions for room overlap capacity and trap placement. More info can be found here.
